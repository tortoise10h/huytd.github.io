* Problem
Chứng minh mối quan hệ giữa lượng từ phổ biến và lượng từ tồn tại:

#+BEGIN_SRC 
¬[∀x ∈ X, P(x)] ≡ ∃x ∈ X, ¬P(x)

¬[∃x ∈ X, P(x)] ≡ ∀x ∈ X, ¬P(x)
#+END_SRC

* Chứng minh
Giả thiết:

#+BEGIN_SRC 
¬[∀x ∈ X, P(x)] ≡ ∃x ∈ X, ¬P(x)
#+END_SRC

Với mọi giá trị x ∈ X thỏa mệnh đề P(x), xét một giá trị a tùy ý:

#+BEGIN_SRC 
∀x ∈ X, P(x) → P(a)  (1)

P(a) → ∃x, P(x)      (2)
#+END_SRC

Từ (1) và (2):

#+BEGIN_SRC 
¬[∀x ∈ X, P(x)] → ¬P(a)

¬P(a) → ∃x, ¬P(x)
#+END_SRC
